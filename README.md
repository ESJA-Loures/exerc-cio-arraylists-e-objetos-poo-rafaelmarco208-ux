# 🚀 Exercício: ArrayLists e Objetos (POO)

Este repositório contém a estrutura base para três exercícios de progressão em Java. Completa cada classe conforme os requisitos abaixo.

---

### 1️⃣ Main.java
**Objetivo:** Operações básicas com `ArrayList<String>`.
* Adicionar os nomes: "Joao", "Maria" e "Pedro".
* Listar todos os nomes.
* Alterar "Joao" para "João".
* Remover "Maria".
* Listar novamente para validar.

### 2️⃣ Main2.java
**Objetivo:** Menu interativo com `Scanner`.
* **Adicionar:** Ler nome da tarefa e guardar.
* **Ver:** Listar tarefas existentes (ou avisar se estiver vazio).
* **Remover:** Ler índice e apagar a tarefa correspondente.

### 3️⃣ Main3.java
**Objetivo:** Gestão de Objetos Complexos.
* **Classe dadosAluno:** Implementar o método `toString()` para exibir "Nº | Nome".
* **Main3:** Criar `ArrayList<dadosAluno>` e implementar o menu (Adicionar, Ver, Remover).
* **Nota:** No adicionar, deves instanciar o objeto: `new dadosAluno(nome, numero)`.
